# fb-clone

A simple Facebook-inspired web application built as part of a Git and GitHub learning tutorial. This project is used to practice version control concepts such as branching, merging, pull requests, tags, and collaboration while developing a social media-inspired interface.

## Purpose
- Learn Git and GitHub workflows
- Practice version control and collaboration
- Experiment with branching and merging
- Build a basic Facebook-style UI

> **Note:** This project is created for educational purposes only and is not affiliated with or endorsed by Facebook (Meta).
